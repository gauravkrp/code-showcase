# myApp-platform

myApp platform App

### Requirements

IDE: VS Code is Preferred.
Node: Version 14 or higher.

### Development

1. Run `npm i` or `yarn` to install packages.
2. Run `npm run dev` to start development server.
3. Open http://localhost:3000/ in the browser.
4. Watch terminal for compilation changes as you write code.

### Production

1. Run `npm run lint` to check for any linting errors.
2. Run `npm run build` for a production build.
3. You should also run `npm run lint-format` to automatically check for linting errors, typescript issues and to automatically apply prettier specified formatting (`prettier --write`) to code before pushing any code to git.
